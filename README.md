# Koniya Store

E-commerce platform with filtering functionalities.

## Used tech

-   React
-   TypeScript
-   TailwindCSS
-   Axios
-   Lucide-react librairy (for icons)

## Development

-   `git clone git@github.com:yannick2019/react-ecom.git`
-   `cd react-ecom`
-   `npm install ...to install dependencies`
-   `npm run dev ...to start developing`
